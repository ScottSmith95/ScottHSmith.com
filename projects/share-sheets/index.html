I recreated iOS's "Share Sheets" using CSS and jQuery. Each Sheet is composed of an ordered list and is able to animate in and out of the page. Options are given for opening a profile in several different social apps. This looks best using and is intended for use only on smartphones. Feel free to copy any of the code I've written for personal use, but I'd love it if you would let me know so I can check out your work too! I have provided a Gist with everything you need to get started.
